# 💰 Token Ranking (Gold Layer)

## 1. 개요 (Overview)
`token_ranking.py`는 Silver 레이어의 토큰 흐름 데이터(`token_flow`)를 분석하여, 이더리움 네트워크에서 **실제 유기적인 성장(Organic Growth)을 보이는 인기 토큰**을 발굴하는 Gold 레이어 파이프라인입니다.

단순 거래량 순위의 허점인 '자전거래(Wash Trading)'와 '기관 노이즈'를 제거하여 대중의 실제 관심도를 정밀하게 측정합니다.

---

## 2. 핵심 설계 의도 (Design Intent)

### 🌱 유기적 거래량(Organic Volume) 산출
*   **워시 트레이딩 감지:** 동일 토큰에 대해 짧은 시간 내에 A → B → A 순환 거래가 발생하는 패턴을 감지합니다.
*   **허수 제거:** 감지된 자전거래 물량은 랭킹 산정 시 제외하여, 인위적으로 부풀려진 거래량을 걷어냅니다.

### 👥 개인 홀더(Individual Holders) 중심 지표
*   **기관 노이즈 필터링:** `whale_analysis`와 동일한 로직을 적용하여, 거래소 지갑이나 DEX 유동성 풀 주소를 제외한 **개인지갑(`Unknown`)** 간의 이동만 활성 주소 수로 집계합니다.
*   **관심도 측정:** 실제 사용자들이 토큰을 주고받는 활동성을 기반으로 '대중의 투심'을 파악합니다.

### ⚖️ 스테이블코인 분리 분석
*   **알트코인 포커스:** 거래량의 대부분을 차지하는 USDT, USDC, DAI 등을 필터링하여, 이들에 가려진 유망한 알트코인들의 랭킹을 별도로 제공합니다.

---

## 3. 주요 분석 지표 (Metrics)

*   **Organic Score:** `유기적 거래량(50%) + 유기적 거래 횟수(30%) + 개인 활성 주소 수(20%)`를 결합한 종합 점수입니다.
*   **Wash Ratio:** 전체 거래량 대비 자전거래로 의심되는 물량의 비율로, 토큰의 신뢰도를 판단하는 척도로 활용됩니다.
*   **New Holders:** 해당 날짜에 처음으로 토큰을 수신한 개인지갑 수를 카운트하여 '신규 유입' 정도를 측정합니다.

---

## 4. 생성 테이블 (Output Tables)

| 테이블명 | 주요 내용 | 활용 방안 |
| :--- | :--- | :--- |
| **`token_ranking`** | 토큰별 유기적 거래량, 점수, 순위 | 인기 토큰 발굴 및 트렌드 분석 |
| **`dex_protocol`** | 유니/스시스왑 등 DEX별 거래 활성도 | 프로토콜별 점유율 모니터링 |
| **`new_holders`** | 토큰별 신규 유입 주소 수 및 순위 | 초기 펌핑 및 대중 유입 감지 |
| **`wash_trade`** | 자전거래 의심 주소 쌍 및 거래량 | 스캠 토큰 필터링 및 데이터 클렌징 |
| **`hourly_heatmap`** | 시간대별 토큰 거래량 변화 | 토큰별 주요 매매 시간대 파악 |

## 5. 실행 방법 (Usage)

```bash
# 특정 날짜의 토큰 랭킹 분석 및 스테이블코인 제외 리포트 출력
uv run src/gold/token_ranking.py --date 2026-05-01 --top-tokens 30 --summary
```

## 6. 향후 확장 가능성
*   **가격 데이터 연동:** 거래량 변화와 가격 변동률을 결합하여 '가성비 좋은(Volatility vs Volume)' 토큰 탐지.
*   **카테고리별 랭킹:** DeFi, NFT 관련 토큰, 밈코인 등 섹터별 세부 랭킹 제공.
