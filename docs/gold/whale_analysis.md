# 🐋 Whale Analysis (Gold Layer)

## 1. 개요 (Overview)
`whale_analysis.py`는 Silver 레이어의 고래 거래 데이터(`whale_txns`)를 가공하여, 이더리움 생태계 내 **거대 자산가(Whale)들의 행동 패턴을 심층 분석**하는 Gold 레이어 파이프라인입니다.

단순한 거래량 집계를 넘어, **기관/거래소 지갑을 제외한 순수 개인지갑(Individual Whale)**의 움직임을 추적하여 실제 온체인 시장의 매집/분산 시그널을 포착하는 것을 목적으로 합니다.

---

## 2. 핵심 설계 의도 (Design Intent)

### 🎯 개인지갑(EOA) 집중 분석
*   **노이즈 제거:** 바이낸스, 유니스왑 등 거래량이 압도적인 기관 지갑을 `address_category == 'Unknown'` 필터를 통해 제외합니다.
*   **타겟팅:** 정체를 알 수 없는 거대 지갑들만 추려내어 진짜 '고래'들의 활동을 현미경처럼 들여다봅니다.

### 📊 고래 심리 및 포지션 파악
*   **포지션 정의:** 순이동량(`net_flow_eth`)에 따라 고래를 `ACCUMULATING`(매집), `DISTRIBUTING`(분산), `NEUTRAL`(중립) 상태로 분류합니다.
*   **타임라인 추적:** 시간에 따른 포지션 변화를 기록하여, 고래가 언제 매수로 전환했는지 또는 매도로 돌아섰는지를 감지합니다.

### 🚨 거래소 유출입 시그널
*   **CEX 입금:** 고래가 거래소로 코인을 보낼 때(잠재적 매도 시그널)를 포착합니다.
*   **CEX 출금:** 고래가 거래소에서 코인을 뺄 때(장기 보유/매집 시그널)를 포착합니다.

---

## 3. 주요 생성 테이블 (Output Tables)

| 테이블명 | 주요 내용 | 활용 방안 |
| :--- | :--- | :--- |
| **`top_whales`** | 당일 활동이 가장 활발한 개인지갑 리스트 및 스코어 | 거대 고래 명단 관리 및 순위 |
| **`exchange_flows`** | 고래 주소별 거래소 입출금 합계 | 시장 매수/매도 압력 파악 |
| **`position_timeline`** | 고래 주소별 시간 순 누적 순이동량 | 고래의 포지션 전환 시점 포착 |
| **`hourly_pattern`** | 시간대별(0~23시) 고래 활동량 집계 | 고래들의 주요 활동 시간대 분석 |
| **`alert_events`** | 대규모 단건 이동, 포지션 전환 등 핵심 이벤트 통합 | 슬랙/푸시 알림 서비스 연동 |

---

## 4. 데이터 품질 보장 (Data Quality)
이 분석은 Silver 레이어의 품질 검사(`check/whale_txns_check.py`)를 통과한 데이터만을 사용합니다.
*   **기술적 무결성:** 소수점 정밀도 유지 및 유효하지 않은 조인 결과 필터링.
*   **현실적 보정:** 소액 이동으로 인한 포지션 노이즈 방지 (0.1 ETH 미만은 중립 처리 등).

---

## 5. 실행 방법 (Usage)

```bash
# 특정 날짜의 고래 분석 실행 및 요약 리포트 출력
uv run src/gold/whale_analysis.py --date 2026-05-01 --top-n 50 --summary
```

---

## 6. 향후 확장 가능성
*   **토큰 고래 연동:** ETH뿐만 아니라 스테이블코인(USDT/USDC) 고래의 움직임과 교차 분석.
*   **라벨링 고도화:** Unknown 주소 중 특정 거래소와의 밀접한 관계를 가진 주소를 AI로 추가 분류.
